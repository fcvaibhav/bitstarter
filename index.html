Features:

  1. Index any video with customizable parameters
  2. Extract to excel to analyse/visualize data
  3. Run filters to play selected parts of the clip only
  4. Key feature: Light and fast perfromance

Applications:

  1. Sports: Can be used by sports clubs, coaches, leagues, betting sites, and media companies to convert match video to data. The indexed video can be run as per custom data filter. E.g. To see the portions of the football match clip where the right back started the move "AND" it finished with a shot on goal. It can be used for many sports like Tennis, Hockey, and Basketball. E.g. in Tennis if you want to get detailed statistics and index the clip to play only the portion where a particular player came to the net and lost the point
  2. Retail/Marketing: Can be used to record the retail data and index it. Later on, you can customize and play with video to understand the changes in store layout in time and correlate it with sales and promotion data to study the impact of store layout
  3. Education: Indexed lectures can help in saving time of students and generating good data about an overview of how much width/deph of a particular topic in present in the video library
